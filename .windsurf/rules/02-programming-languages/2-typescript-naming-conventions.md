globs: **/*.ts, **/*/.tsx
---

Classes and Interfaces:
- Use PascalCase for names
- Use nouns or noun phrases

Functions and Methods:
- Use camelCase for names
- Use verbs for actions
- Use nouns for value-returning
- Prefix booleans with is, has, should
- No anemic models
