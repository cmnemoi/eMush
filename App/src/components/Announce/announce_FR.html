Le lien sera disponible sur Eternal-Twin & un code sera distribué le jour du lancement (Uniquement aux joueurs qui participent au vaisseau test de l'Alpha. En privé ou sur le discord.) Pour faire partie du vaisseau test alpha, vous aurez besoin d'un compte sur le site Eternal-Twin. Vous aurez besoin de vous connecter avec votre compte Twinoid. Les comptes Hammerfest & Dino-Parc fonctionnent aussi.

Pour lier son compte Eternal-Twin à votre compte Twinoid, vous aurez besoin d'être ami avec Demurgos sur Twinoid. Vous devrez accepter son application qui demande l'accès à toutes les données venant du site Twinoid. Cela permettra de garder l'ancienne progression de tous vos jeux.
